Differences in Audacity_Audiotouch

Key Combinations which differ from standard Audacity 1.2.6 in Audacity_Audiotouch:
  • Ctrl + S = Save as WAV file
  • Ctrl + Shift + S = Save Selection as WAV file
  • Ctrl + G = Save as OGG file
  • Ctrl + Shift + G = Save Selection as OGG file
  • Ctrl + M = Save as MP3 file
  • Ctrl + Shift + M = Save Selection as MP3 file
Note: All these commands will also include exiting Audacity and responding "No" to save changes dialog.

  • Ctrl + P = Save Project
  • Ctrl + Shift + P = Save Project As
